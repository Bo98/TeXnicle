TeXnicle 2.0

Finally, and just in time for the holidays, here is the final release of version 2.0 of TeXnicle. Work has already started on version 2.1, but there will be bug fix releases in the interim.

  • Added an xelatexmk engine.
  • The 'Open Console' engine setting is now ignored when in 'Live Update' mode.
  • Fixed a bug in the built-in symbol palette: the preview images for the lowercase and capital Xi had the same filename on disk.
  • Fixed various other bugs relating to stability.
