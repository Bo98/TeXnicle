NOTE: If you are using TeXnicle on a 32-bit machine, please contact support at bobsoft-mac.de and let us know. The current intention is to drop support for 32-bit machines in the next release!

This release contains various minor bug fixes, optimisations and the ability to set your own colors for the document outline view.

Some changes of note

  • The Spelling navigator no longer shows the possible corrections in the table. It turns out that gathering the possible corrections for all misspelled words is very computationally expensive for any reasonable sized documents. The corrections are just presented now on demand.
